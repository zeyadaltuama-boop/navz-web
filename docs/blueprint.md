# **App Name**: RideShift

## Core Features:

- Passenger Ride Request: Allows passengers to request a ride by specifying pickup and dropoff locations. Calculates fare estimates and displays available drivers with ETAs.
- Driver Availability Toggle: Drivers can toggle their availability status to online or offline, signaling their readiness to accept ride requests.
- Real-Time Ride Tracking: Provides real-time updates on the ride status and driver location via a map interface for both passengers and drivers.
- Payment Processing: Integrates payment gateway(s) to facilitate payment via card and other digital wallets.
- Post-Ride Feedback: Enables passengers to rate drivers and provide feedback after each ride.
- Admin Ride Management: Allows admin users to manage drivers, passengers, rides, and subscriptions through tables, filters, and dashboards.
- Driver Compliance Check: AI-powered tool for automatically checking driver-submitted documents against requirements and highlighting any problems for admin attention.

## Style Guidelines:

- Primary color: Deep sky blue (#498af0) to invoke trust and safety.
- Background color: Light gray (#f0f2f5) for a clean, modern interface.
- Accent color: Green (#3cb371) for positive actions and key elements.
- Body and headline font: 'Inter' sans-serif for a modern, machined look
- Use flat, minimalist icons to represent ride types, locations, and user actions.
- Map-centric design for passenger app with intuitive controls and clear information hierarchy.
- Subtle transitions and animations to provide visual feedback during ride booking and status updates.
